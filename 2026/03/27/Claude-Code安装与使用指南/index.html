客户端选择

安装完成后，有两种使用方式：

方式 工具 特点
CLI（命令行） CC Switch 轻量，支持多 Agent 管理
GUI（图形界面） Tokenicode 轻量级图形界面，配合 CC Switch 做 Agent 管理

推荐组合： Tokenicode + CC Switch，既有友好的图形界面，又能灵活管理多个 Agent。

总结

平台 核心依赖 安装命令
Mac Homebrew + Node.js 国内镜像脚本
Windows Git + Node.js winget + npm

两个平台最终都通过 npm install -g @anthropic-ai/claude-code 完成 Claude Code 的安装，步骤简洁，上手门槛低。
