###类

  • 类的概念

  • 类有很多概念还是没有吃透和理解

  • 类有属性

类的属性就如同可口可乐的饮料配方

类可以实例化

  • 利用饮料的配方在生产线上生产出饮料产品

  • 生产的饮料产品有着相同的配发(属性)

类实例化后也有属性

  • 生产线生产出的饮料，需要对其进行包装，包装就是实例的属性

类实例化后 肯定是希望能做一些事情

  • 这是就可以在类里定义函数，让这个类去做一些事情

  • 比如，可乐可以用来喝，用来赠送等等行为
