爬虫-怎么爬静态网站

爬静态网站主要分为两部分：

  1. 爬静态网站的文字

  2. 爬静态网站的图片

[TOC]

爬文字

思路

  1. 用requests模块得到网站的HTML

  2. 用BeautifulSoup模块得到HTML的正则文本

  3. 用find或者find_all函数从正则文本中得到自己想要的

  4. 用repalce去除不需要的字符

爬图片

思路

  1. 用requests模块得到网站的HTML

  2. 用BeautifulSoup得到HTML的正则文本

  3. 用find函数从正则文本中得到自己想要的，比如关键词img

  4. 利用urllib模块下载

  5. 利用for语句下载所有图片
