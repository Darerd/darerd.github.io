LaTex:参考文献引用的方法

发表于 2019-02-20 分类于 产品经理学技术

LaTex排版时，引入参考文献的方法

引入参考文献主要分为两种：

  1. 不使用BibTex

  2. 使用BibTex

这里我们主要介绍更为复杂的第二种：使用BibTex引入参考文献

利用BibTex的方法引入文献：

分为以下四步：

  1. 用LaTex编译写好的 .tex 文件，这是会生成一个 .aux文件，这是告诉BibTex将使用那些引用

  2. 用BibTex编译 .aux文件（有些仅编译 .bib文件，也可以）

  3. 再次用LaTex编译 .tex 文件，这是文档已经插入了参考文献，但是引用的编号不对

  4. 最后在用LaTex编译一次 .tex文件，这是会自动生成正确的引用顺序（引用顺序是与在文章中引用文献的先后顺序有关）

如果你采用的是TexStudio作为编译器，具体的操作方法如下：

  1. 按F6编译写好的 .tex 文件

  2. 按F8编译生成的 .aux 文件

  3. 按F6编译写好的 .tex 文件

  4. 按F6编译写好的 .tex 文件

以上就是使用LaTex排版时，遇到的引入参考文献的解决方法
