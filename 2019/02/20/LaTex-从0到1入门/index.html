第一步

到这一步，我们知道LaTex是一种电子排版系统，是公认的数学公式排得最好的系统，主要作用是让你专注于写作，得到高质量的排版输出。
​ 学习LaTex，那就需要能够编辑它和翻译它的工具—LaTex编译器(小编自己起的名字😆 )
​ 我试了很多编译器，如：Lyx、TeXworks、WinEdt，用来用去最后最爱的还是TexStudio。
​ 我建议新手可以先从线上编辑器(如：Overleaf) 先学习。

一篇详细介绍LaTex编译器的帖子
