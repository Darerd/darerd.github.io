作为一名产品经理，日常免不了要和大量的文档打交道，比如：需求文档、竞品分析、MRD、PRD等等

而不甘心的你，在学习了Markdown之后，不在为格式发愁了，大大提高了输出效率

进门右转，Markdown入门

利用Markdown写作基本帮我们解决了格式排版的问题(如果对格式要求非常高，可以出门左转，学些Latex排版)
