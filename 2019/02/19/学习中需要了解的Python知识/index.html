2. input() 和 raw_input() 命令的区别

Python2.x 中 input() 相等于 eval(raw_input(prompt)) ，用来获取控制台的输入。

raw_input() 将所有输入作为字符串看待，返回字符串类型。

而 input() 在对待纯数字输入时具有自己的特性，它返回所输入的数字的类型（ int, float ）。
